<h2>Text comparison tool</h2>
This code allows to compare two texts.<br>

This tool allows you to upload large text files (hundreds of pages) and find pairs of passages in reasonable time (few minutes at most).<br>

Online version: https://versuser-n5tyntby6aud5yryzwrdgf.streamlit.app/

<h2>Instructions to install</h2>

* Create a virtual environment (pyenv, conda, etc.) with a Python version >= 3.11.5
* Run `pip install -r requirements.txt` to install the required libraries
* Run `streamlit run App_st.py` to launch the application
